## 🔬 Research Directions

- **Diffusion & Visual Generation** — acceleration, controllable generation, object removal, poster synthesis
- **Multimodal Reasoning & MoE** — routing analysis, cross-modal intervention, expert activation
- **LLM Controllability & Steering** — unified frameworks for parameter dynamics, behavioral evaluation
- **AI Safety** — multimodal adversarial robustness, safety benchmarking
